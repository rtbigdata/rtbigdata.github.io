    Data and Maps!

    Working on some projects to explore data and maps using modern open source tools.

    The first of these is done using Leaflet.js. It is a map of the San Diego area showing median year built of homes by Census Block Group. Darker areas in this choropleth map indicate neighborhoods likely to contain older homes. The data is from the 2014 American Community Survey.
